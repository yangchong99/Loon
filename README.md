# Loon 资源与配置集锦

> **Loon** 是一款功能强大的 iOS 网络代理工具。本仓库旨在整理、优化并分享适用于 Loon 的插件、脚本及配置规则，助你打造极致的网络环境。

-----

## 📑 核心功能

本仓库主要涵盖以下四大模块：

  * **🛡️ 规则分流**：精简高效的广告拦截与全球加速规则。
  * **🔌 功能插件**：一键增强应用功能（如去广告、深色模式切换等）。
  * **📜 自动化脚本**：集成各类签到、任务处理的 JavaScript 脚本。
  * **🎨 视觉优化**：针对 Loon 界面及通知样式的个性化配置。

-----

## 🚀 快速开始

### 1\. 导入配置

复制下方链接，在 Loon 配置文件管理中选择“从 URL 下载”：
`https://raw.githubusercontent.com/yangchong99/Loon/main/default.conf`

### 2\. 添加插件

进入 **插件 (Plugins)** 界面，点击 `+` 号并粘贴你需要的 `.plugin` 链接。推荐必备项：

  * **广告屏蔽**：精选国内外主流 App 广告去重。
  * **流媒体增强**：优化 Netflix/YouTube 观看体验。

-----

## 📂 目录结构

```bash
├── Script/        # 核心脚本 (.js)
├── Plugin/        # 功能插件 (.plugin)
├── Rule/          # 分流规则 (.list)
└── Config/        # 完整配置文件 (.conf)
```

-----

## 💡 注意事项

1.  **证书配置**：使用重写 (Rewrite) 或脚本功能前，请确保已安装并信任 **CA 证书**。
2.  **按需取用**：建议根据实际需求添加插件，避免因插件过多导致设备耗电或网络延迟。
3.  **同步更新**：建议开启“资源解析器”自动更新，以获取最新的规则与脚本。
